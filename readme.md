## Variables to configure on .env ##

```AWS_DEFAULT_REGION=
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
S3_BUCKET=
AGG_ID=
SFTP_SERVER=
SFTP_USER=
SFTP_PASS=
EMAIL_RECIPIENT=
EMAIL_ERROR_RECIPIENT=
JOB_INTERVAL=1  #value in minutes
ZENITH_MASK=v2

```
## How to run ##
Check run.sh script
First Param - Command
Second param - Environment
